1. Требования
PHP 8.2 или выше
Composer

2. Установка
Выполните команды в терминале из папки проекта: Установка зависимостей:

composer install

Настройка окружения:
Файл .env убедитесь, что настройки базы данных выглядят так:

DB_CONNECTION=sqlite
DB_DATABASE="C:/путь_к_проекту/database/database.sqlite"

(Замените путь на актуальный для вашей папки или оставьте пустым, если файл лежит в стандартном месте).

3. Подготовка базы данных:
Генерация ключа и наполнение базы таблицами и тестовыми статьями:

php artisan key:generate
php artisan migrate:fresh --seed

4. Запуск сервера:

php artisan serve

Сайт будет доступен по адресу: http://127.0.0.1:8000
